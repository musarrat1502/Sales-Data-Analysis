# Sales Data Analysis

This project performs Exploratory Data Analysis (EDA) on the Superstore Sales dataset using Python.

## Objective
To analyze sales data and identify key business insights.

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Key Insights
- Technology category generated the highest revenue.
- West region contributed the most to overall sales.
- Consumer segment has the highest purchasing volume.
- Sales distribution is positively skewed (more low-value transactions).

## Steps Performed
- Data Loading
- Data Inspection
- Missing Value Check
- GroupBy Analysis
- Data Visualization
